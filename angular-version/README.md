# Angular Audio Transcription App

## Setup Instructions

This Angular version requires dependency installation before it can run:

```bash
# Install Angular dependencies
npm install

# Run the development server
ng serve

# Or if ng CLI is not available globally:
npx ng serve
```

## Prerequisites

- Node.js 16+ 
- Angular CLI (optional, can use npx)

## Dependencies Required

The following Angular packages will be installed automatically:
- @angular/core
- @angular/common  
- @angular/platform-browser
- @angular/platform-browser-dynamic
- @angular/forms
- @angular/router
- @angular/compiler
- rxjs
- zone.js
- tslib

## Build and Run

```bash
# Development mode
npm run start

# Production build
npm run build

# Run tests
npm run test
```
